Rexster is a multi-faceted graph server that exposes any Blueprints graph (e.g. TinkerGraph, Neo4j, OrientDB, DEX, and Sail RDF Stores) through several mechanisms with a general focus on REST. This HTTP web service provides standard low-level GET, POST, PUT, and DELETE methods, a flexible extensions model which allows plug-in like development for external services (such as adhoc graph queries through Gremlin), server-side “stored procedures” written in Gremlin, and a browser-based interface called The Dog House. Rexster Console makes it possible to do remote script evaluation against configured graphs inside of a Rexster Server.1

Rexster Kibbles is a collection of various Rexster server extensions.

Please join the Gremlin users group at http://groups.google.com/group/gremlin-users for all TinkerPop related discussions.
